## Interpretation

- The real 650M ESM2 load, download/cache path, CUDA forward path, frozen embedding cache, head-only training, metric calculation, and checkpoint writing all worked on the local RTX 2080.
- `batch_size=1` and `max_len=64` were used to keep the smoke test within 8 GB GPU memory.
- The `position_col` path was not exercised in this run because `--use_mock_data` currently creates records without positions. Unit tests cover position pooling and CSV validation; a future smoke fixture with explicit positions would better exercise the production feature path.
